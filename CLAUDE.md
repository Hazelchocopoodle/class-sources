## Structure

- `html01/` - HTML 기초
- `html02/` - HTML 심화
- `html03/` - CSS 선택자, 스타일링
- `html04/` - Flexbox 레이아웃, Form 요소
- `js01/` - JavaScript 기초 (DOM, 이벤트)
- `js02/` - JavaScript 심화 (동적 요소 생성)

- 모든 파일은 단일 HTML 파일로 작성 (인라인 CSS/JS 사용)
