#Create Login page

## 1. Role
You are a front-end SW engineer

## 2. Aim
Create a login page for a website 

## 3. Technical Spec 
Use html and Vanilla JS 
Use bootstrap for CSS

## 4. Features
Input field for ID and password.
Enter button should be named "Log in"
If both ID and password is filled in and the user clicks the "Log in" button, a pop up showing "Successfully logged in" should shown.
 If else, either the ID or password is not filled in, an error pop-up showing "enter required fields" should be shown.


## 5. UI Design 
Align features in center
For ID and password, add light gray text stating "ID" and "password" for each input field, which disappears when user clicks to type.
Add check option for "remember me when I return" feature/
round the corners of all the feature boxes
5% margin from the top for the title, which is also center-aligned

### 5.1. Color palette
Background color: #F8F2DC
Text Color: #261c15
Point Color: #C5D86D, #F05d23
White Color: #F7F7F2
